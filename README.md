# Explanation Stability Analysis under Dataset Shift

This project implements a modular research pipeline for evaluating the stability of model explanations under dataset shift. The framework studies how explanations generated by different modelвЂ“explainer pairs change when the data distribution changes, while controlling for prediction drift.

The project focuses on two shift settings:

- Covariate shift
- Label shift

## Fixed ModelвЂ“Explainer Pairs

The project is organized around the following four fixed modelвЂ“explainer pairs:

- **RF + SHAP**
- **RF + Permutation Importance**
- **MLP + SHAP**
- **MLP + LIME**

These pairs are the core analytical units of the project. All explanation files, stability calculations, plots, and summary tables should follow this structure.

## Objective

The main goal is to quantify explanation stability under controlled dataset shifts using an **Explanation Stability Index (ESI)**.

For local explanation methods, ESI compares baseline explanations and shifted explanations using:

- rank stability
- top-k feature overlap
- magnitude similarity

The framework also applies a **prediction-stability filter** so that explanation drift is measured only on instances whose prediction probabilities remain sufficiently stable.

For **RF + Permutation Importance**, which is a global explanation method, stability is treated as a global comparison rather than a strictly instance-level comparison.
